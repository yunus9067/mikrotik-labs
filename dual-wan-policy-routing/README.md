# MikroTik Dual WAN Policy Routing and Failover

## Overview
This project demonstrates the implementation of Dual WAN, Policy Routing, Routing Mark, Static Route, and Failover using MikroTik CHR.

The configuration allows:
- PC1 to use ISP1 as the primary gateway
- PC2 to use ISP2 as the primary gateway
- Automatic failover if one ISP connection goes down

---

# Features
- Dual WAN Configuration
- Static Routing
- Policy Based Routing
- Routing Mark using Mangle
- Automatic Failover
- NAT Masquerade
- DHCP Server
- Traffic Separation per Client
